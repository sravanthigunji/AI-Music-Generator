# 🎵 AI Music Generator

AI-based music generation project developed using Python, TensorFlow, LSTM neural networks, and MIDI music data.

## 🚀 Features

* MIDI preprocessing using music21
* Note sequence extraction
* LSTM deep learning model
* AI-generated music creation
* MIDI output generation
* Streamlit web interface

## 🛠 Technologies Used

* Python
* TensorFlow / Keras
* music21
* NumPy
* Streamlit

## 📂 Project Files

* preprocess.py
* train.py
* generate.py
* app.py
* music_model.h5
* generated_music.mid

## ▶️ Run Project

### Install Dependencies

```bash
pip install tensorflow music21 numpy streamlit
```

### Train Model

```bash
python train.py
```

### Generate Music

```bash
python generate.py
```

### Run Streamlit UI

```bash
streamlit run app.py
```

## 🎼 Output

The AI model generates new music sequences and converts them into MIDI files which can be played or downloaded.
